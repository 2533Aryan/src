Name: ARYAN RAVAL
Project: Generate Art using Neural Networks.

------------------------------------------------------------------------------------------------------

Description:

- This project will generate various art using Neural Network.

------------------------------------------------------------------------------------------------------

Prerequisite:

- Need "gen-art.core" (core.clj) main file  :  for drawing art by connecting network, math, & quil.
- Need "gen-art.math" (math.clj) file       :  for performing math functions.
- Need "gen-art.network" (network.clj) file :  for our neural networks.
- Need "quil.core" (import quil) library    :  for drawing (displaying) our art.

------------------------------------------------------------------------------------------------------

Instruction:

- It will take some time (maybe 5-10 minutes) to generate art. So please don't stop the execution.
- Run core.clj file for drawing art (it will connect other files).
